# Windows Downloads Sentinel v1.0

## Overview
A "Performance-First" background utility that autonomously organizes your Downloads folder. Designed for gamers, it features a "Gamer Mode" that pauses all execution when high CPU load or full-screen applications are detected.

## Features
- **Performance First**: Zero resource usage during gaming.
- **Privacy Airlock**: Sensitive files (financial, legal) stay local.
- **Hybrid AI**: Uses Rules (Tier 1), Gemini Cloud (Tier 2), and Local AI (Tier 3).
- **Ghost UI**: System Tray interface.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure `config/secrets.json` with your API keys.
3. Run: `python src/main.py`
